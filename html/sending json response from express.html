From a Node.js express application app.get() function, how do I send a response in json?

In a Node.js Express application, you can send a JSON response using the res.json() method within an app.get() route handler. Here's how you can do it:

Basic Example:

javascript
Copy
Download
const express = require('express');
const app = express();

app.get('/api/data', (req, res) => {
    const data = {
        message: 'Hello, World!',
        status: 'success',
        items: [1, 2, 3]
    };
    
    // Send the response as JSON
    res.json(data);
});

app.listen(3000, () => {
    console.log('Server is running on port 3000');
});

Key Points:

  1. res.json() automatically:

    • Sets the Content-Type header to application/json.

    • Converts the JavaScript object to a JSON string using JSON.stringify().

  2. You can also include a status code:

    javascript
    Copy
    Download
    res.status(200).json({ message: 'Success' });

  3. If you want to send a simple JSON response without an object:

    javascript
    Copy
    Download
    res.json({ success: true });

Example with Dynamic Data:

javascript
Copy
Download
app.get('/api/user/:id', (req, res) => {
    const userId = req.params.id;
    const user = {
        id: userId,
        name: 'John Doe',
        email: 'john@example.com'
    };
    
    res.json(user);
});

When you access the route (e.g., http://localhost:3000/api/data), the response will be:

json
Copy
Download
{
    "message": "Hello, World!",
    "status": "success",
    "items": [1, 2, 3]
}
